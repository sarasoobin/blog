# Chapter 4. SQL 고급

## 1. SQL 내장 함수

SQL 함수는 DBMS가 기본으로 제공하는 **내장 함수**와 사용자가 직접 만드는 **사용자 정의 함수**로 나뉜다. 내장 함수는 단일 값을 반환하며 `SELECT`, `WHERE`, `UPDATE` 절 등에서 사용 가능하다.

### 숫자 함수

수학적 계산을 돕는 함수들이다.

* **ABS(숫자):** 절댓값 계산 (예: `ABS(-78)` → 78)
* **ROUND(숫자, m):** 반올림. m은 반올림 기준 자릿수
* **CEIL / FLOOR:** 각각 숫자보다 크거나 같은 최소 정수, 작거나 같은 최대 정수를 반환

### 문자 함수

문자열을 다룰 때 사용하는 함수들이다.

* **REPLACE:** 특정 문자를 다른 문자로 치환
* **CONCAT:** 두 문자열을 하나로 합침
* **SUBSTR:** 지정한 위치에서 원하는 길이만큼 잘라냄
* **LENGTH vs CHAR_LENGTH:** `LENGTH`는 바이트 수를(한글 UTF-8 기준 3바이트), `CHAR_LENGTH`는 순수 문자 수를 반환

### 날짜/시간 함수

* **STR_TO_DATE:** 문자열을 날짜형으로 변환
* **DATE_FORMAT:** 날짜형을 원하는 문자열 형식으로 변환
* **ADDDATE:** 날짜에 특정 기간을 더하거나 뺌
* **SYSDATE() vs NOW():** `NOW()`는 쿼리 시작 시점의 시간을, `SYSDATE()`는 함수가 호출되는 바로 그 순간의 시간을 반환

## 2. NULL 값 처리

`NULL`은 '0'이나 공백과는 다른, **"아직 지정되지 않은 값"**을 의미한다.

* **NULL의 연산:** `NULL + 숫자`는 무조건 `NULL`
* **집계 함수와의 관계:** `SUM`, `AVG` 계산 시 `NULL` 행은 제외된다. 모든 행이 `NULL`이면 결과는 `NULL`이지만, `COUNT`는 `0`을 반환
* **IFNULL(속성, 값):** 속성값이 `NULL`이면 지정한 '값'으로 대치하여 출력

## 3. CASE WHEN

프로그래밍의 `IF-THEN-ELSE`와 같다. 특정 조건에 따라 값을 분류하거나, 집계 함수 내부에 써서 조건별 합계를 구할 때 사용한다.

```sql
CASE
    WHEN 조건1 THEN 결과1
    WHEN 조건2 THEN 결과2
    ELSE 기본값
END AS 별칭
```

## 4. 부속질의 (Subquery)

하나의 SQL 문 안에 중첩된 또 다른 SELECT 문을 말한다. 위치에 따라 이름이 달라진다.

1. **중첩질의 (WHERE 절):** 데이터를 필터링할 때 사용. `IN`, `EXISTS`, `ALL`, `ANY` 등의 연산자와 함께 쓴다.
2. **스칼라 부속질의 (SELECT 절):** 결과로 딱 하나의 값(단일 행, 단일 열)만 반환해야 한다.
3. **인라인 뷰 (FROM 절):** 서브쿼리 결과를 마치 임시 테이블처럼 사용한다.

## 5. 윈도우 함수 (Window Function)

**윈도우 함수**는 `GROUP BY`와 비슷해 보이지만 결정적인 차이가 있다. `GROUP BY`는 행을 합쳐버리지만, 윈도우 함수는 **원본 행의 개수를 유지하면서 그룹 계산 결과만 각 행에 표시해 준다**.

### 기본 문법

```sql
SELECT 함수명() OVER (PARTITION BY 컬럼명 ORDER BY 컬럼명) FROM 테이블명;
```

* **PARTITION BY:** 계산을 수행할 그룹을 나눈다.
* **ORDER BY:** 그 그룹 안에서 계산을 수행할 순서를 정한다.

### 주요 함수 유형

* **순위 함수:** `RANK()`, `DENSE_RANK()`, `ROW_NUMBER()`
* **집계 함수:** `SUM()`, `AVG()`, `MAX()`, `MIN()` 등을 `OVER`절과 함께 사용하여 누적 합계 등을 구할 수 있다.
* **분석/값 함수:** `LEAD()` (다음 행 값), `LAG()` (이전 행 값) 등으로 시계열 분석에 유용하다.

윈도우 함수를 쓰면 복잡한 셀프 조인(Self Join) 없이도 순위를 매기거나 누적치를 구할 수 있어 코드가 간결해진다.
